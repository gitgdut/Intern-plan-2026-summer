# AI Agent 如何拥有支付能力？

## 2026/07/08

**AI Agent 如何拥有支付能力 — 核心观点提炼**

**分享人**：Jack Xiao（蚂蚁背景，负责生态与合作）

\---

**一、为什么 AI Agent 需要自己的支付体系**

传统支付系统是为**人**设计的，但 Agent 的支付场景完全不同：

\- **Agent 没有身份证** → 开不了银行账户
\- **微支付场景**：调一次 API 可能只要 $0.001–0.01 → 传统手续费比金额还高
\- **结算太慢**：T+1、T+3 对需要实时执行的 Agent 不可接受
\- **不能每笔都让人确认**：Agent 要 7×24 小时自主工作，人在环内就失去意义

Agent 的支付逻辑是 **"意图发现 → 一系列操作"**，不是简单的"付钱→收货"。比如写一份研究报告，需要调用大模型 API、爬付费数据、买图片——是一连串微支付的组合。

\---

**二、他们做的产品：一条完整的 Agent 支付协议**

不是单一工具，而是一整套体系，覆盖支付的每个环节：

***\*Identity（身份）****

• 模块: **Identity（身份）**

• 说明: 给 Agent 一个"身份证"，让它能被识别和授权

***\*Budget（预算/约束）****

• 模块: **Budget（预算/约束）**

• 说明: 人设定花钱上限和条件，Agent 在框内自由花

***\*Risk Control（风控引擎）****

• 模块: **Risk Control（风控引擎）**

• 说明: 防止幻觉导致的乱花钱，金融场景容错率为零

***\*Audit（审计追溯）****

• 模块: **Audit（审计追溯）**

• 说明: 每笔交易可查可追溯

***\*Reversal（可撤销）****

• 模块: **Reversal（可撤销）**

• 说明: 一键撤销交易，不需要改代码

**产品形态**：

\- **钱包**：人+Agent 共管，不是纯 Agent 自主控制（一和零之间取中间态）
\- **Agent Card**：一次性预付费卡，设定额度锁死，用完即废；正在跟 Visa 合作推出可重复使用的版本
\- **Monetize 平台**：开发者把 API/MCP/技能上架，Agent 自动发现并付费调用
\- **Payment Link**：适合电商等金额不固定的场景
\- **Embedded Payment Protocol**：基于 HTTP 的 x402 协议——人签一次授权，Agent 在预算内自由操作，最后批量结算，无需逐笔人工确认

\---

**三、支付通道**

\- **加密侧**：基于 **Base 链 + USDC**（合规考虑，USDC 在美国合规）
\- **法币侧**：通过 **Stripe** 入金，转为平台积分（credential）
\- 两条路并存，Agent 用哪条取决于场景

\---

**四、关键落地场景**

1. **抢红包 Agent**：春节期间做的社交实验，Agent 自动发现红包并抢
2. **百度智能云合作**：帮开发者把 API/MCP 货币化，Agent 自动发现并付费调用
3. **任务平台**：Agent 完成任务 → 自动收款
4. **订票 Agent**：跟大麦合作，Agent 帮用户抢演出票
5. **VC 知识付费**：某硅谷 VC 把行业见解 skill 化，Agent 付费调用获取报告
6. **工具箱变现**：独立开发者把能力变成可被 Agent 调用的付费服务


\---

**五、核心判断**



>  "Agent commerce 是不可逆的未来。OpenAI 的 operator 产品出来后，我们更坚定了这个判断。"

\- 支付是 Agent 经济的基础设施，绕不过去
\- 微支付是小切口，但随着信任建立，大额支付也会跟上来（类似支付宝早期没人敢存大钱）
\- 合规是必选项——所以他们跟 Visa 深度合作，未来会做 KYC/KYB
\- **Base 链发展最快的原因是 Coinbase 行动力快 + 合规底子好**，以太坊主网受基金会流程拖累
\- Agent 不会彻底取代人，但会替代大部分重复性工作——而这些工作过程中一定会涉及钱

\---

**六、一句话总结**

**给 AI Agent 装上一个"钱包 + 身份证 + 预算约束 + 风控"的完整支付体系，让它能像人一样花钱——但不能乱花。**
